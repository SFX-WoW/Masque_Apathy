IMPORTANT:
- You MUST have the most recent revision of Button Facade installed in order to use this skin.

About:
- This is a skin for Button Facade that will add some custom button styles to the supported action bars.

Installation:
- Download the package.
- Unzip the package to your World of Warcraft/Interface/AddOns/ directory.
